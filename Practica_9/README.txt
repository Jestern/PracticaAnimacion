Practica 9 - Animación por ordenador - 16/17
Autor: Elías Méndez García

Practica realizada con Unity 5.6.1f1, para abrir los ficheros se necesita como mínimo esta versión.

Para activar las animaciones se usan las teclas, Ctrl para la animación de agacharse y T para la animación del saludo.
